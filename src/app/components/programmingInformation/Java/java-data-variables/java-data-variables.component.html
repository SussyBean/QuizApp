Java променливи и тип-данни

Какво е променлива в Java?

Променливата в Java е контейнер с данни, който съхранява стойностите на данните по време на изпълнение на Java програма. На всяка променлива се присвоява тип данни, който обозначава вида и количеството стойност, която може да съдържа. Променлива е име на място в паметта на данните. Променливите на Java имат основно три типа: локални, инстанционни и статични.

Типовете данни се характеризират с:

  • Име;

  • Размер (колко памет заемат);

  • Стойност по подразбиране (default value);
