Функции в JavaScript

Какво е функция в JavaScript?




Функциите са едни от основните градивни елементи в JavaScript.

Те са подобни на процедурите – набор от изрази, които изпълняват задача или изчисляват стойност, но за да се квалифицира процедурата като функция, тя трябва да вземе някакъв вход и да върне изход.
За да може една функция да се използва,то тя трябва да бъде дефинирана в обхвата, от който тя се извиква.






Как се дефинира функция?

Дефиницията на функция (наричана още декларация на функция или израз на функция) се състои от ключовата дума на функцията, последвана от:

 - Името на функцията.
 - Списък с параметри на функцията, затворени в скоби и разделени със запетаи.
 - Изявленията на JavaScript, които дефинират функцията, затворени в къдрави скоби.



На долния пример можем да разгледаме проста функция, която ни връща квадрата на число


function square(number) ❴

return number * number;

❵


Функцията square(квадрат) приема един параметър,с име number (число).
Функцията се състои от едно твърдение (statement), което казва да върне параметъра на функцията (number), умножен по себе си. Именно със специалната дума return се връща квадрата на това число.
Параметрите по същество се предават на функциите по стойност, т.е ако в тялото на функция параметъра,подаден на функцията, присвои напълно нова стойност, то тази промяна не се отразява глобално, а вътре в самата функция.



Но когато като параметър се подава обект и вътре в функцията се променят свойствата на този обект,то тези промена СА видими извън функцията, както е показано в следващия пример:



function myFunc(theObject) ❴

theObject.make = 'Toyota'; Параметърът взима стойност Toyota

❵

var mycar = ❴  make: 'Honda', model: 'Accord', year: 1998  ❵ ;

var x, y; x = mycar.make; //  Променливата х присвоява стойността Honda

myFunc(mycar);

y = mycar.make; // Променливата y присвоява стойността Toyota

                             // (свойството make се променя от функцията и може да се използва)
